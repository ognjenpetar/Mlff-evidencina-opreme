SVRHA APLIKACIJE:
-----------------
Web aplikacija za evidenciju i praćenje opreme na MLFF (Multi-Lane Free-Flow)
naplatnim portalima. Koristi Supabase cloud backend za neograničen kapacitet
podataka i pristup sa bilo kog uređaja.


GLAVNE FUNKCIONALNOSTI:
=======================

1. UPRAVLJANJE LOKACIJAMA
-------------------------
- Dodavanje novih lokacija (naplatni portali)
- Unos GPS koordinata (latitude/longitude)
- Upload fotografije lokacije (do 50MB)
- Opis lokacije
- Pregled svih lokacija na interaktivnoj mapi
- Izmena i brisanje lokacija

2. EVIDENCIJA OPREME
--------------------
Svaki komad opreme ima sledeće podatke:
- Tip opreme (VDX, VRX, Antena, Switch, TRC, TRM, intel, jetson, Wi-FI)
  * Predefinisani tipovi + mogućnost unosa custom tipova
- Inventarski broj (jedinstvena oznaka)
- Status: Aktivna, Na servisu, Neispravna, Neaktivna, Povučena
- Sub-lokacija: Gentri ili Ormar
- Proizvođač i Model
- Serijski broj
- IP adresa i MAC adresa
- X/Y/Z koordinate u centimetrima
- Datum postavljanja
- Datum isteka garancije
- Ime instalatera i testera
- Fotografija opreme (do 50MB)
- PDF dokumentacija (neograničen broj fajlova, do 50MB po fajlu)
- Napomene

3. BULK OPERACIJE (NOVO u v5.0)
-------------------------------
- Masovna selekcija opreme (checkbox-ovi)
- Masovna promena statusa više komada odjednom
- Masovno brisanje opreme
- Toolbar sa bulk akcijama

4. NAPREDNA PRETRAGA (NOVO u v5.0)
----------------------------------
- Filter po lokaciji
- Filter po tipu opreme
- Filter po statusu
- Filter po sub-lokaciji
- Filter po proizvođaču
- Filter po datumu instalacije (od-do)
- Filter po datumu isteka garancije (od-do)
- Kombinovanje više filtera

5. NOTIFIKACIJE (NOVO u v5.0)
-----------------------------
- Upozorenja za istek garancije (30 dana unapred)
- Upozorenja za planirano održavanje (7 dana unapred)
- Upozorenja za opremu koja je predugo na servisu (14+ dana)
- Badge sa brojem aktivnih notifikacija
- Dropdown lista sa detaljima

6. QR KOD SISTEM
----------------
- Automatski generisan QR kod za svaki komad opreme
- QR kod sa MLFF logom
- Skeniranjem se otvara kompletan izveštaj bez prijave
- Štampanje nalepnica sa QR kodom
- Preuzimanje QR koda kao PNG slike

7. ANALITIKA I DASHBOARD (PROŠIRENO u v5.0)
-------------------------------------------
Osnovni KPI-jevi:
- Ukupan broj opreme
- Broj aktivne opreme
- Broj opreme na servisu
- Ukupni troškovi održavanja

Napredni KPI-jevi (NOVO):
- Dostupnost (% aktivne opreme)
- Garancijska pokrivenost (% pod garancijom)
- Trošak po opremi (prosečan)
- Prosečna starost opreme

Dashboard filteri (NOVO):
- Period (7, 30, 90, 365 dana)
- Lokacija
- Tip opreme
- Status

Grafikoni:
- Status Pie Chart (sa drill-down)
- Type Bar Chart (sa drill-down)
- Cost Trend Line Chart
- Age Distribution Chart
- Location Bar Chart (stacked)

Lokacijska analitika (NOVO):
- Interaktivna mapa sa markerima
- Boja markera po statusu opreme
- Popup sa detaljima lokacije

Export (NOVO):
- Export u PDF
- Export u Excel

8. NAVIGACIJA I STRUKTURA
-------------------------
- Breadcrumb navigacija
- Bočni panel sa hijerarhijskim prikazom
- Status indikatori (zeleno=aktivna, žuto=servis, crveno=neispravna, sivo=neaktivna)
- Hash-based routing za shareable URL-ove

9. PRETRAGA I FILTRIRANJE
-------------------------
- Globalna pretraga
- Filter po tipu, statusu, lokaciji
- Instant rezultati

10. PRAĆENJE STATUSA
--------------------
Pet statusa opreme:
- AKTIVNA - oprema radi normalno (zelena)
- NA SERVISU - oprema je na popravci/održavanju (žuta)
- NEISPRAVNA - oprema ne funkcioniše (crvena)
- NEAKTIVNA - privremeno van upotrebe (siva) - NOVO u v5.0
- POVUČENA - trajno uklonjena iz upotrebe (tamno siva)

11. ISTORIJA PROMENA (AUDIT LOG)
--------------------------------
- Automatsko beleženje svih promena
- Stare i nove vrednosti
- Datum, vreme i korisnik
- Ime polja koje je promenjeno

12. MODUL ZA ODRŽAVANJE/SERVIS
------------------------------
- Tip servisa (Preventivni, Korektivni, Inspekcija, Zamena dela, Kalibracija)
- Datum servisa
- Ime servisera
- Opis radova
- Troškovi (EUR)
- Datum sledećeg servisa

13. IZVEŠTAJI I ŠTAMPANJE
-------------------------
- Print izveštaja za opremu
- Print izveštaja za lokaciju
- In-app preview izveštaja


TEHNIČKA ARHITEKTURA:
=====================

FRONTEND:
- HTML5, CSS3, JavaScript (ES6+)
- Single Page Application (SPA)
- Leaflet.js za mape
- Chart.js za grafikone
- QRCode.js za QR kodove
- Font Awesome za ikone
- Vite za build

BACKEND:
- Supabase (PostgreSQL + Storage + Auth)
- Row Level Security (RLS)
- Google OAuth autentifikacija

HOSTING:
- GitHub Pages (besplatno)
- Unlimited bandwidth

CLOUD STORAGE:
- Supabase Storage (1GB besplatno)
- Do 50MB po fajlu
- Public URL-ovi za fajlove

KAKO POKRENUTI:
===============

1. SUPABASE SETUP:
   - Kreiraj projekat na https://supabase.com
   - Pokreni SQL migracije iz supabase/migrations/ folder
   - Konfiguriši Google OAuth
   - Kopiraj API credentials

2. LOKALNI DEVELOPMENT:
   npm install
   cp .env.example .env
   # Popuni .env sa Supabase credentials
   npm run dev

3. PRODUCTION BUILD:
   npm run build
   # Output: dist/ folder

4. DEPLOYMENT:
   Push na GitHub → GitHub Pages automatski hostuje
